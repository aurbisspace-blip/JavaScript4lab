## 📋 Цель работы

Познакомиться с классами и объектами в JavaScript, научиться создавать классы, использовать конструкторы и методы, а также реализовать наследование и функции-конструкторы.

- [x] Реализован метод `getInfo()` для вывода информации о предмете
